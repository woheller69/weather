RadarWeather позволяет следить за погодой в интересующих вас городах и местах.

Данные о погоде предоставляются через бесплатный API от OpenWeatherMap. Поскольку новый одновызовный API допускает только 1000 вызовов в день, каждый пользователь должен ввести свой собственный бесплатный ключ OpenWeatherMap (https://home.openweathermap.org/users/sign_up).
Данные для карты осадков получаются через бесплатный RainViewer API.

Особенности:
- Текущая погода
- Почасовые прогнозы на 48 часов / 3 часа на 5 дней
- 8-дневный прогноз
- Осадки каждый следующий час
- Карта осадков
- Поиск по радиусу
- Без рекламы

Минимальные разрешения:
RadarWeather требует только минимального количества разрешений, а именно разрешения ИНТЕРНЕТ. Это разрешение необходимо для выполнения HTTP-запросов к удалённым серверам для получения данных о погоде.
